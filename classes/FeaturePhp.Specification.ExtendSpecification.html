      Settings for specifying an extendable file.

      Extended specifications are used by a \FeaturePhp\Generator\ExtendGenerator. For concrete implementations, see \FeaturePhp\Specification\ChunkSpecification or \FeaturePhp\Specification\TemplateSpecification. The extended specification settings follow the structure:

      • root (object)
        • mayCreate (bool) - whether this specification may create the specified file, if this is false and the file does not exist, the specification is ignored
